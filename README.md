# dbwrite

## NOTICE: Work in progress

Easily work with Appwrite databases.

> Will soon available at `dbwrite.js.org`. Waiting for approval.

We used [Docsify](https://docsify.js.org) for our documentation.

## Installation

### Bun

```bash
bun add appwrite
```

### Npm

```bash
npm i appwrite
```
