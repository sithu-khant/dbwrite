Documentation at: [dbwrite.js.org](https://dbwrite.js.org/#/).

We used [Docsify](https://docsify.js.org) for our documentation and subdomain from [JS.org](https://js.org).

## Installation

### Bun

```bash
bun add dbwrite
```

### Npm

```bash
npm i dbwrite
```
